Movie Booking Website UI

Project Overview

BSR Movies is a dark-themed movie ticket booking website designed to provide users with a smooth and engaging experience. The project showcases an interactive interface where users can browse now showing, upcoming, and next shows, explore movie details, and easily navigate for booking. The design focuses on visual impact with bold imagery, easy-to-use navigation, and clear categorization of movies, creating a premium and user-friendly look.

Key Features

  • Hero section with featured movie banner including description, genre, cast, and trailer option
  • Tabbed navigation for "Now Showing," "Next Show," and "Coming Show"
  • Card-based movie highlights with posters and details
  • Navigation links for About, Business, Ticket booking, and Contact
  • Footer with quick links & contact info
  • Social media integration for connectivity

Technologies Used

  • Figma (UI design, prototyping, component creation)
  • Design systems & grids for layout consistency
  • Typography & color styles for theme management
  • Interactive prototyping for navigation flow

My Role & Learnings

  • Designed and developed the UI/UX layout with a dark-themed, premium look
  • Implemented interactive elements such as carousels and tab navigation
  • Focused on user-centric design for easy booking and browsing
  • Learned to structure large-scale UI with clear hierarchy and reusability
  • Improved skills in responsive layouts and modern UI patterns
